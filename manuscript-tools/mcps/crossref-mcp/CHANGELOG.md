# Changelog

## [0.1.0] - 2025-02-15

### Added
- Initial release with Crossref API integration
- `search` tool for querying Crossref by query, author, title, year
- `get_metadata` tool for retrieving paper metadata by DOI
- `get_bibtex` tool for fetching BibTeX citations
- `find_published_version` tool for locating published versions of working papers
