# Changelog

## [0.1.0] - 2025-02-15

### Added
- Initial release with BibTeX file operations
- `parse_bib` tool for parsing `.bib` files into structured indices
- `scan_bare_citations` tool for detecting bare citations in `.qmd`/`.tex` files
- `rekey_entry` tool for rekeying BibTeX entries to match project conventions
- `add_entry` tool for adding entries to `.bib` files
- `suggest_replacement` tool for generating proper citation commands
